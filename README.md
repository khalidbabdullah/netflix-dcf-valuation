# Netflix DCF Valuation Model

A comprehensive discounted cash flow (DCF) valuation model for Netflix Inc. (NFLX) built in Excel. This model projects free cash flows, calculates enterprise and equity value, and includes sensitivity analysis to assess valuation under different scenarios.

## 📊 Model Overview

**Valuation Approach:** Discounted Cash Flow (DCF)  
**Company:** Netflix Inc. (NFLX)  
**Tool:** Microsoft Excel  
**Key Output:** Intrinsic equity value per share with sensitivity tables

## 🔑 Key Features

- **Revenue Projections:** Multi-year forecast based on subscriber growth and ARPU assumptions
- **Operating Model:** Detailed income statement projections with margin analysis
- **Free Cash Flow Calculation:** NOPAT-based FCF with working capital adjustments
- **WACC Calculation:** Weighted average cost of capital using CAPM
- **Terminal Value:** Perpetuity growth method for terminal value estimation
- **Sensitivity Analysis:** Two-way data tables testing WACC and growth rate assumptions
- **Scenario Analysis:** Bull, base, and bear case valuations

## 📈 What This Model Does

This model values Netflix by:
1. Projecting revenue growth based on streaming subscriber trends
2. Modeling operating expenses and margins
3. Calculating unlevered free cash flows
4. Discounting cash flows at the company's WACC
5. Adding terminal value to determine enterprise value
6. Converting to equity value and per-share valuation

## 🛠️ Model Structure

The Excel workbook contains:
- **Assumptions:** Key inputs for revenue growth, margins, WACC components
- **Income Statement:** Projected P&L through forecast period
- **Cash Flow:** Free cash flow calculation and DCF valuation
- **Valuation Summary:** Enterprise value, equity value, and price per share
- **Sensitivity Tables:** Impact of varying discount rate and terminal growth assumptions
